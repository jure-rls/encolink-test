# encoLink Master Example

Reference C implementation of the encoLink master-side protocol for
AksIM-2 encoders.

## Which version do I need?

| Encoder firmware | Protocol | Branch | Latest release | Status |
|---|---|---|---|---|
| rev 2.x | encoLink v2 | `main` | proto2-1.0 | Active |
| rev 1.x | encoLink v1 | `v1.x` | proto1-1.1 | Legacy |
